# Pre-backend keepsake

This folder is a **historical snapshot** of the project **before** the Express/Postgres backend was added.

- **`frontend-head/`** — React source + Tailwind CRA app (design reference).
- **`root-head/`** — Older root snapshot (may include `node_modules` locally; those are **not** meant to be committed — see root `.gitignore`).

For the version with **student/teacher login wired to the API + heat map**, use:

`keepsake/integrated-frontend-snapshot/` (at repo root).
