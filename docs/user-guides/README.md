# User Guides

This folder contains role-based usage guides and demo scripts for the AIR @TAMGU platform.

## Files

- `instructor-guide.md`: day-to-day instructor workflow.
- `student-guide.md`: what **ABC** (this platform) is for SSI-focused STEM, data literacy and agency, plus day-to-day student workflow.
- `instructor-demo-guide.md`: guided instructor demo script.
- `student-demo-guide.md`: guided student demo script.
