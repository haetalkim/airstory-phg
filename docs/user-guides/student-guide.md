# Student Guide

## Account and Access

- Sign in with your assigned student account.
  - Group 1 demo user: `jiin@tamgu.com`
  - Group 4 demo user: `julia@tamgu.com`
- Students can view analytics and submit edit annotations.

## Core Workflow

1. Log in and open the dashboard.
2. Check your group session data and latest measurements.
3. Switch metrics to understand air-quality behavior over time.
4. Add reflections/notes from your observations.

## Editing Data (Annotation Mode)

- You can propose corrections to measured values in the measurement table.
- Enter a new value and save with an optional note.
- Edited values display with `*`.
- The original sensor value is preserved for auditability.

## Best Practices

- Only submit edits with a clear reason (for example, sensor validation correction).
- Keep notes short and specific.
- Do not use edits to hide outliers; use reflections to explain context.

## What Students Cannot Do

- Students cannot overwrite canonical source data directly.
- Students cannot manage workspace roles or backend configuration.
