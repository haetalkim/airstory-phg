# Student Demo Guide

Use this to demo collaborative student behavior.

## Demo Personas

- Student Group 1: `jiin@tamgu.com` / `password`
- Student Group 4: `julia@tamgu.com` / `password`

## Demo Flow (4-6 minutes)

1. **Login as Group 1 student**
   - Show dashboard load with prefilled measurements.
   - Select a metric and explain trend interpretation.
2. **Submit an edit annotation**
   - Pick one measurement field.
   - Enter a corrected value, add note, and save.
   - Confirm value now shows `*`.
3. **Switch to Group 4 student**
   - Log out and log in as second student.
   - Show different group session context/data.
4. **Compare observations**
   - Contrast Group 1 and Group 4 patterns.
   - Add reflection note about potential causes.

## Talking Points

- Students can contribute corrections responsibly.
- Original values remain preserved.
- Group-based data supports classroom comparison.
