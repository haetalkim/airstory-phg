# Instructor Demo Guide

Use this for a live walkthrough in front of stakeholders.

## Demo Persona

- Instructor: `shim@tamgu.com` / `password`

## Demo Flow (5-7 minutes)

1. **Login**
   - Sign in as instructor.
   - Confirm workspace and role context.
2. **Overview**
   - Point out summary metrics and trend chart.
   - Switch between `PM2.5`, `CO`, `Temperature`, `Humidity`.
3. **Group Comparison**
   - Show sessions for Group 1 and Group 4.
   - Highlight differences in values and heat points.
4. **Edit Annotation Review**
   - Locate a measurement with `*`.
   - Explain original vs edited value and annotation note.
5. **Export**
   - Trigger CSV export.
   - Mention optional Google Sheets export endpoint.

## Talking Points

- Multi-role workflow (instructor + students).
- Traceability via edit annotations.
- Team-ready workspace model.
- Data export for analysis and reporting.
